## Installation

Install the Goodfire SDK using pip:

```bash
pip install goodfire
```

## Initialize the SDK

Start by importing and initializing the Goodfire client:

```python
from sdk import goodfire

client = goodfire.Client('YOUR_API_KEY')
```
